# Social Media Dashboard Project

## Project Overview
This project is a Social Media Dashboard built using React.  
It simulates core features of modern social media platforms such as user profiles, post feeds, notifications, and theme switching.

## Objectives
- Build a component-based React application
- Practice state management using hooks
- Implement routing using React Router
- Separate data logic using a mock API
- Create a responsive and user-friendly interface

---

## Features
- User Profile section
- Post feed with multiple posts
- Notifications panel
- Dark / Light mode toggle
- Responsive layout
- Persistent theme using localStorage

---

## What I Learned
1. Breaking UI into reusable React components  
2. Managing state using `useState` and `useEffect`  
3. Client-side routing using React Router  
4. Separating data logic using a mock API  
5. Using custom hooks (`useLocalStorage`)
